3 easy ways to get your software live — from free instant hosting to professional VPS deployment.
FREE · Easiest
▲
Vercel (Recommended)
Deploy in 2 minutes. Free SSL, global CDN, custom domain support. Best for static HTML/JS apps.
1
Go to vercel.com → sign up free
2
Click "Add New Project" → drag & drop your folder
3
Vercel auto-detects HTML → click Deploy
4
Your site is live at yourname.vercel.app
5
Add custom domain: mercurypayroll.com.np
FREE · GitHub
🐙
GitHub Pages
Free hosting directly from your GitHub repository. Perfect for sharing and version control.
1
Create a GitHub account at github.com
2
Create new repository → upload all files
3
Go to Settings → Pages → select main branch
4
Live at: yourusername.github.io/mercury-payroll
5
Connect custom .com.np domain in DNS settings
PRO · Full Backend
🖥️
VPS / DigitalOcean
For full production with Node.js backend, PostgreSQL database, and real payroll processing.
1
Get a VPS from DigitalOcean or AWS (~$6/mo)
2
Install Node.js, PostgreSQL, Nginx
3
Upload files via FTP or Git clone
4
Point your domain DNS to server IP
5
Install SSL with Let's Encrypt (free)
⚡ Quickest way — Vercel CLI (2 minutes)
Step 1 — Install Vercel CLI
npm install -g vercel # Install once

cd mercury-payroll vercel # Login → confirm → done! # Your app is live at: https://mercury-payroll.vercel.app
Step 3 — Add your custom .com.np domain
